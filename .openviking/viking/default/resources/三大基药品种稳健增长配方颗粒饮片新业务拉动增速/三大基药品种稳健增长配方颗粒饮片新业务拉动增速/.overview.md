# 三大基药品种稳健增长配方颗粒饮片新业务拉动增速

## 详细描述

### 三大基药品种稳健增长配方颗粒饮片新业务拉动增速_1.md 三大基药品种稳健增长配方颗粒饮片新业务拉动增速_1.md

本文是信达证券发布的佐力药业2024年业绩及2025年前景的公司点评研究报告。报告提供了详实的财务数据和业务分析：

**核心业绩数据**：公司2024年实现营业收入25.80亿元，同比增长32.81%；归母净利润5.05亿元，同比增长31.79%。2025年Q1预计归母净利润1.78-1.89亿元，同比增长24.99%-32.71%，显示增长势头持续强劲。

**三大基药品种表现**：乌灵胶囊、灵泽片、百令片作为公司核心产品保持稳健增长，其中乌灵胶囊销售数量增长22.62%，灵泽片销售数量增长23.17%，体现了传统优势产品的持续生命力。

**新业务拉动增速**：中药饮片营业收入增长45.82%，中药配方颗粒增长145.34%，成为公司增速的主要驱动力，展现了新业务板块的巨大潜力。

**长期增长目标**：公司实施股权激励和员工持股计划，2025-2027年净利润增长率目标分别不低于30%、66%、110%，体现了管理层对公司发展的高度信心。

本报告适合医药行业投资者、分析师及相关从业人士，需要具备医药产业知识和财务分析能力。

### 三大基药品种稳健增长配方颗粒饮片新业务拉动增速_2.md 三大基药品种稳健增长配方颗粒饮片新业务拉动增速_2.md

本文是信达证券发布的医药行业公司点评研究报告，重点分析乌灵系列产品及中药饮片业务的增长表现，从多个维度深化对公司发展动力的理解。

**三大基础药品种分析**：详细阐述了乌灵胶囊、灵泽片、百令片等核心产品的市场表现和增长驱动因素，分析其在公司收入中的重要地位。

**新业务板块评估**：重点阐述配方颗粒和中药饮片作为新业务板块对公司增速的拉动作用，分析这两个业务的快速放量增长潜力和市场前景。

**盈利能力提升机制**：评估精细化管理对盈利能力的提升效果，包括成本控制、运营效率优化等方面的具体措施和成果。

**管理层信心指标**：关注公司股份回购行为所反映的持续经营信心，这是评估管理层对公司前景看好程度的重要参考。

本报告面向医药行业投资者、基金经理和证券分析师，需要具备医药产业基本知识和财务分析能力，是进行投资决策的重要参考资料。
