## Detailed Description

### 上周研究报告_41/

本子目录是一份医药行业研究报告的综合索引库，系统汇集了2017年至2025年间发布的医药及医疗健康领域的深度研究报告。该目录涵盖多个研究维度，包括医药上市公司个股分析、行业周报、投资策略、细分领域研究等，覆盖A股、港股及海外制药企业的全面分析。

**主要内容特点**：
- 时间跨度长（2017-2025年），提供历史纵深视角
- 覆盖医药产业链全景，包括创新药、医疗器械、CXO等细分领域
- 包含上市公司个股深度分析，适合精选投资
- 涵盖出海战略研究，关注中国医药企业国际化进程
- 提供行业周报等定期更新资料，便于持续跟踪

**适用人群**：医药行业投资者、基金经理、研究人员、产业分析师、医疗从业人员

### 精选行业报告_43/

本子目录是一份系统化的医药行业研究报告汇编，专注于2020-2024年期间医药产业各细分领域的深度分析。该目录通过精选报告的方式，为用户提供高质量的行业研究资源，涵盖行业周报、基金持仓分析、政策动态、创新药械研发进展、疫苗血制品、肿瘤治疗等多个专题。

**主要内容特点**：
- 时间聚焦于2020-2024年，反映近期行业发展趋势
- 多维度分析框架：包括行业周报、基金持仓、政策分析等
- 重点关注创新药械研发进展，追踪行业技术创新动向
- 专题深化研究：疫苗血制品、肿瘤治疗等热点领域
- 投资评级体系完整，便于投资决策参考
- 政策动态实时更新，帮助把握政策导向

**适用人群**：医药行业投资者、基金经理、产业分析师、医疗从业人员、政策研究人员
