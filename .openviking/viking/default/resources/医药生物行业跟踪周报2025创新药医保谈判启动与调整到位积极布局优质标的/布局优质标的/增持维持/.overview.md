# 增持维持

本目录是东吴证券研究所发布的医药行业周报系列文档集合，采用增持维持的投资评级体系，为医药板块投资者提供周期性的市场跟踪、政策分析和投资建议。文档全面覆盖医药股收益表现、医保谈判政策、创新药研发进展、行业监管动态、估值水平分析等核心内容。适合医药行业投资者、基金经理、证券分析师和相关从业者阅读，需具备基本的医药产业知识和证券投资理解。**核心关键词**：医药行业、创新药、医保谈判、市场估值、研发进展、投资评级、板块追踪。

## Detailed Description

### 本周及年初至今各医药股收益情况_1/

本子目录是东吴证券研究所发布的医药行业周报系列核心数据库，全面追踪和分析医药上市公司的股票收益表现、新药研发进展、药品审批动态和政策监管信息。涵盖A股和港股医药企业50余家，包括制药、生物科技、医疗服务、医药流通等多个细分领域。文档采用数据表格形式，详细记录周收益率、年初至今收益率、市值、临床试验阶段、药品靶点等关键指标。通过量化数据和定性分析相结合的方式，帮助投资者快速定位高潜力医药企业和风险预警信号。适合医药行业投资者、基金经理、证券分析师等专业人士进行投资决策和行业研究。**核心关键词**：医药股收益、行业周报、新药研发、药品审批、投资分析、估值追踪。

### 附录_7/

本附录汇集了药品分类参考指南和证券研究报告的相关规范文档，主要涵盖生物制品、中药制剂的分类标准，以及投资评级体系和免责声明。这些文档为制药行业从业者、药品注册申报人员、监管部门工作人员和专业投资者提供了详细的参考资料。内容涉及新药分类、临床优势评估、技术改进标准、给药途径改变等关键概念，同时规范了投资评级的基准和方法，明确了增持、维持、减持等评级的定义和适用条件。附录还包含免责声明和法律合规说明，确保研究报告的专业性和规范性。适用于需要了解药品分类标准、进行新药申报、或进行医药行业投资分析的专业人士。
