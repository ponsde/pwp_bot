# 增持维持

本目录是一份系统性的医药行业投资分析资料库，汇集了医药板块市场表现追踪、医保基金运行监测、创新药研发进展评估及上市公司财务指标分析。内容涵盖医保政策影响、行业监管动态、估值评估和个股投资建议，采用数据图表结合的形式呈现。适合医药行业投资者、基金经理、证券分析师及医药研究人员参考，帮助理解市场趋势和投资机会。**核心关键词**：医药股、医保基金、创新药、板块分析、财务指标、投资评级、估值分析。

## Detailed Description

### 本周及年初至今各医药股收益情况_1/

本目录汇集了东吴证券研究所发布的医药行业周报系列文档，全面追踪医药板块上市公司的股票收益表现、医保基金运行状况及创新药研发进展。涵盖CRO/CDMO、生物制药、化学制药、医疗服务、药品零售等多个产业链环节，提供详细的周度、月度及年初至今的收益数据对比。文档面向医药行业投资者、基金经理、证券分析师及研究人员，是进行医药板块投资决策和行业研究的重要参考资料。

### 研发进展与企业动态_3/

本目录是一份医药行业研究资料集合，重点追踪医药子板块市场表现、个股动态和创新药研发进展。包含东吴证券研究所发布的医药行业周报、生物制品分类体系资料以及制药企业的研发管线数据。涵盖中药、化学药、生物制品等多个领域，涉及肿瘤、免疫、心血管、代谢等治疗方向。适合医药行业投资者、分析师、制药企业、监管部门及医药研究人员了解市场动态、评估投资机会和研发进展。**核心关键词**：医药板块、个股表现、创新药、改良药、研发进展、药品注册、临床试验、上市申报、集采降价、医保政策。

### 附录_7/

本附录汇集了中医药行业监管分类标准和东吴证券研究报告的规范性文档。主要包括中药新药的四大分类体系（创新型、改良型、古代经典名方、同名同方）的详细阐述，以及证券研究报告的免责声明和投资评级标准说明。该附录面向医药行业从业者、监管人员、投资分析人士和证券客户，帮助理解中药产品的监管框架和投资评估体系。**核心关键词**：中药分类、新药审评、投资评级、风险提示、监管标准。
