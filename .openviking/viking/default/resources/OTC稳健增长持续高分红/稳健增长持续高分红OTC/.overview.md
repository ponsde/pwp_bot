# 稳健增长持续高分红OTC

本目录包含西南证券对江中药业2024年年报的全面投资分析报告。通过建立三大业务板块（OTC、处方药、健康消费品）的增长假设，详细展示了公司2024-2027年的财务预测、估值分析和投资评级。核心发现显示江中药业保持稳健增长态势（年均增速13.6-13.7%），毛利率稳定在63%以上，同时实施高分红政策（股利支付率92-96%）。适合证券分析师、基金经理、投资者等专业人士了解江中药业的长期投资价值和财务健康状况。

## 详细描述

### 稳健增长持续高分红OTC_1.md 稳健增长持续高分红OTC_1.md

本文档是江中药业2024年年报的投资分析点评，通过建立三大业务板块的增长假设进行财务预测。OTC板块预计2025-2027年保持15%稳定增速，毛利率维持72%；处方药板块受集采降价影响2025年增速为0%，之后恢复5%增长，毛利率39%；健康消费品及其他板块预计20%高速增长，毛利率37%。核心预测显示公司整体收入将从2024年44.4亿元增长至2027年65.2亿元，年均增速约13.7%，毛利率保持在63%以上水平，体现稳健增长特征。适合需要了解江中药业业务结构和增长驱动因素的投资分析师和基金经理。

### 稳健增长持续高分红OTC_2.md 稳健增长持续高分红OTC_2.md

本文档是江中药业2024年年报的财务分析点评，包含详细的财务预测与估值数据。涵盖三个核心财务报表：利润表展示2024-2027年营业收入、成本、利润等指标的实际值与预测值；现金流量表分析经营、投资、筹资活动的现金流动情况；资产负债表反映公司资产、负债及股东权益结构。此外还包含财务分析指标体系，涵盖成长能力、获利能力、营运能力和资本结构四个维度，以及PE、PB、PS等估值指标和每股收益、股利等关键指标。关键发现显示公司保持稳健增长态势，销售收入年均增长约13.6%，净利率维持在18-19%，ROE逐年提升至25.24%，同时保持高分红政策，股利支付率超过92%。

### 稳健增长持续高分红OTC_3.md 稳健增长持续高分红OTC_3.md

本文档是上市公司财务分析报告，展示了2024年至2027年的详细财务预测和估值分析。包含四个核心部分：利润表分析显示营业收入稳健增长，从2024年的444.35亿元增至2027年的652.53亿元，净利润相应增长；现金流量表反映经营活动现金流充足，投资和筹资活动保持稳定；资产负债表表明公司无息债务、资产结构健康、流动性充足；财务指标体系涵盖成长能力、获利能力、营运能力和资本结构。关键特点包括毛利率稳定在63%以上，ROE和ROA持续提升，股利支付率高达92-96%体现持续高分红政策，PE估值从18.16倒数至12.34显示增长潜力，EV/EBITDA从10.53下降至5.99表明投资价值。
