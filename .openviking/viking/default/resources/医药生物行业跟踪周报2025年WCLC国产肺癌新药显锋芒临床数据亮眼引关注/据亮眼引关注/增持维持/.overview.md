# 增持维持

本目录是东吴证券医药行业研究团队发布的周期性跟踪报告集合，专注于医药行业的市场动态、投资机会和风险评估。内容涵盖医药指数表现、创新药临床进展、企业财务指标、新药审批动态及细分领域投资建议。通过系统追踪A股和港股医药上市公司的股票收益、研发管线、临床试验数据和监管政策变化，为投资者提供全面的行业分析和投资决策支持。**核心关键词**：医药周报、创新药、临床试验、市场分析、投资评级、估值追踪、监管动态、财务指标、CXO、GLP1。**适用人群**：医药行业投资者、证券分析师、基金经理、研究人员及医药从业者。

## Detailed Description

### 本周及年初至今各医药股收益情况_1/

本目录汇集了东吴证券研究所发布的医药行业周报系列文档，是追踪医药上市公司股票表现的核心资料库。内容重点关注A股和港股50余家医药企业的收益表现、周期性波动、新药研发进展、临床试验数据及药品审批动态。涵盖创新药企业、CXO服务商、生物制品企业和医疗器械公司等多个细分领域，提供多维度的收益数据、药品注册信息、临床管线进展和市场动态分析。文档包含具体的财务指标、估值追踪、新药临床数据（如iza-bren、HARMONi、HLX43等）、行业监管政策更新，以及创新药、CXO、GLP1等多个投资方向的具体标的选择建议。适合医药行业投资者、证券分析师、基金经理、研究人员和医药从业者参考，需具备基础医药知识和投资背景。

### 附录_7/

本附录是东吴证券行业跟踪周报的重要补充资料，包含中药新药分类管理标准和投资研究报告的合规声明。主要涵盖中药新药的四大类别定义、审批标准、管理规范，以及证券研究报告的免责声明和投资评级体系。中药新药分类部分为医药产业分析提供了规范性参考，投资评级体系明确了报告中各类评级的含义和适用范围。本附录适用于医药行业从业人员、投资研究机构、监管部门和证券投资者，作为中药产业分析和投资决策的参考依据，同时确保投资建议的合规性和透明度。
