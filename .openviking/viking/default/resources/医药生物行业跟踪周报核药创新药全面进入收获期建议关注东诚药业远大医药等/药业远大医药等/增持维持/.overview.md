# 增持维持

本目录是东吴证券研究所发布的医药行业周报系列文档集合，采用增持维持评级体系，重点追踪医药股市场表现、产业动态和投资机会。核心关注领域包括放射性核素偶联药物（RDC）、创新药研发进展、核药产业链发展，以及医药板块估值变化。文档面向医药行业投资者、分析师、基金经理和研究人士，提供市场分析、产业洞察和投资配置建议。**关键词**：医药股、核药、RDC、创新药、投资配置、产业动态、估值分析、监管政策。

## Detailed Description

### 本周及年初至今各医药股收益情况_1/

本目录是东吴证券研究所发布的医药行业周报系列，主要统计和分析医药上市公司的股票收益表现、新药研发进展、临床试验动态和产品管线信息。涵盖A股和港股市场50余家医药企业，包括药明康德、百济神州、恒瑞医药等龙头企业，以及创新药、CXO、核药、生物制药等细分领域的代表性公司。文档采用数据表格和专业分析相结合的方式，为投资者、基金经理、证券分析师等专业人士提供市场动态、产业前沿信息和投资决策参考。核心内容包括周度收益排行、年初至今累计收益、新药临床进展追踪、产品管线更新等。

### 行业洞察与监管动态_4/

本目录汇集了东吴证券研究所关于医药行业的专业研究报告和监管政策指南。内容涵盖医药板块市场表现追踪、估值分析、个股涨跌统计、行业风险评估，以及药品和疫苗的分类监管标准。该资料库适合证券投资者、基金经理、医药企业研发人员、监管部门工作人员和行业分析师使用，提供市场监测、投资参考和政策理解的全面支持。重点关注医药板块的相对估值变化、龙头企业表现、细分领域发展趋势，以及政策风险因素对产业的影响。

### 附录_7/

本附录汇集了药品分类标准和证券投资评级体系的核心参考资料。内容涵盖生物制品和中药制剂的详细分类定义、监管标准，以及投资评级的具体方法论和免责声明。本附录适用于制药企业研发人员、药品监管部门、证券投资机构和行业分析人士，帮助相关人员准确理解药品注册分类、评估投资价值和规范风险管理。核心内容包括生物制品分类体系、中药改良标准、新药申报流程、相对评级方法论、风险提示和免责条款。
