# 本周及年初至今各医药股收益情况_1

本目录是东吴证券研究所发布的医药行业周报系列，主要统计和分析医药上市公司的股票收益表现、新药研发进展、临床试验动态和产品管线信息。涵盖A股和港股市场50余家医药企业，包括药明康德、百济神州、恒瑞医药等龙头企业，以及创新药、CXO、核药、生物制药等细分领域的代表性公司。文档采用数据表格和专业分析相结合的方式，为投资者、基金经理、证券分析师等专业人士提供市场动态、产业前沿信息和投资决策参考。
