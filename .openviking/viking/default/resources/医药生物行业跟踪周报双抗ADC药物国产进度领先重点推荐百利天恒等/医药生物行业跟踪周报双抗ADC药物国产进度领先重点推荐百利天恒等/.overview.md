# 医药生物行业跟踪周报双抗ADC药物国产进度领先重点推荐百利天恒等

本目录包含东吴证券2024年12月29日发布的医药生物行业周报系列文档，重点聚焦双抗ADC（抗体偶联药物）领域的国产创新药发展进展。报告通过行业动态监测、技术进展分析和重点企业评估，阐述国内ADC药物在全球竞争中的领先地位。核心内容涵盖医药指数走势、双特异性抗体技术突破、国产创新药管线进展（百利天恒BL-B01D1、康宁杰瑞JSKN-003等）、以及14个细分领域的投资推荐。适合医药行业投资者、基金经理、证券分析师等专业人士阅读，需具备基础医药产业知识和资本市场认知。

### 医药生物行业跟踪周报双抗ADC药物国产进度领先重点推荐百利天恒等_1.md 医药生物行业跟踪周报双抗ADC药物国产进度领先重点推荐百利天恒等_1.md

本文档是东吴证券2024年12月29日发布的医药生物行业周报，采用周期性跟踪研究模式，为投资者提供及时的行业动态和投资建议。

**行业走势分析**：医药指数本周下跌1.91%，其中医疗服务、医药商业等板块跌幅较大，反映出市场对医药板块的短期调整压力。

**双抗ADC药物重点进展**：报告重点突出国产双抗ADC药物的全球领先地位，详细介绍了4款进度最快的国产双抗ADC产品：
- 百利天恒的BL-B01D1
- 康宁杰瑞的JSKN-003
- 其他两款国产产品

这些产品有望在2025年递交上市申请，代表了中国生物制药产业的创新突破。

**最新产业进展**：包括华辉安健HH-003申报上市、科伦药业伊布替尼片获受理、甘李药业胰岛素周制剂进入III期等重要动态。

**细分领域推荐**：报告覆盖创新药、中药、CXO、医疗服务等14个细分领域，提供了百利天恒、信达生物、恒瑞医药等重点推荐标的，帮助投资者全面把握医药生物行业的投资机会。
