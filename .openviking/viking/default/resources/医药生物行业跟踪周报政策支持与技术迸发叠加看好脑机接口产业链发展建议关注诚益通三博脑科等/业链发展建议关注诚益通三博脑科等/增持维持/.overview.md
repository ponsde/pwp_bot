# 增持维持

本目录是东吴证券医药行业研究团队发布的系列周报文档集合，采用增持维持评级策略，专注于追踪医药板块市场动态、产业链发展和投资机会。文档涵盖医药股市场表现统计、脑机接口等前沿技术产业链分析、创新药研发进展、估值指标监测以及具体投资组合建议。适合医药行业投资者、基金经理、产业研究人员以及具备医药产业基础知识的分析师参考。**核心关键词**：医药股、创新药、脑机接口、估值追踪、投资策略、产业链分析、周度表现、药品注册。

## Detailed Description

### 本周及年初至今各医药股收益情况_1/

本目录是东吴证券研究所发布的医药行业周报合集，系统追踪医药上市公司的股票收益表现，并深入分析脑机接口等前沿产业发展动态。包含50余家医药企业的多时间维度收益数据、创新药审批进展、药品注册信息及脑机接口产业政策分析。涵盖生物制药、医疗器械、医药CRO、医疗服务等细分领域。

**主要内容特点**：
- 周度医药股收益排行与市场表现统计
- 年初至今累计收益数据对标分析
- 创新药审批进展与药品注册动态追踪
- 脑机接口产业链深度分析与政策解读
- 医药细分领域（生物制药、医疗器械、CRO等）的专项研究
- 估值指标监测与投资机会识别

**适用人群**：医药投资者、基金经理、证券分析师、产业研究人员、医药企业战略规划部门。

### 附录_7/

本附录汇集了药品分类参考指南和证券研究报告的规范性文档，主要涵盖生物制品、中药制剂的分类标准体系，以及投资评级和免责声明规范。这些文档来源于药品审评中心（CDE）和东吴证券研究部门，为制药企业研发人员、监管部门、投资分析机构和证券投资者提供权威的分类参考和投资评估框架。

**主要内容特点**：
- 新药申报分类标准与指南
- 改良型药物的细分方向与评估标准
- 生物制品与中药制剂的分类体系
- 投资评级体系与相对性质说明
- 证券研究报告的规范性要求与免责声明
- 药品审评中心（CDE）权威分类参考

**适用人群**：制药企业研发人员、药品监管部门、投资分析机构、证券投资者、医药产业政策研究人员。
