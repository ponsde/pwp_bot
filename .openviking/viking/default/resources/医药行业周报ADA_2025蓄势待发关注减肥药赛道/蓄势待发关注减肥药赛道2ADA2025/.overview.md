# 蓄势待发关注减肥药赛道2ADA2025

这是一份关于减肥药物赛道的专业研究报告集合，由爱建证券于2025年6月17日发布。报告系统分析了GLP-1受体激动剂、多靶点激动剂、口服小分子激动剂等减肥药物的最新临床进展，重点关注ADA2025学术大会上的重磅数据公布。内容涵盖全球主流药物（司美格鲁肽、替尔泊肽、Retatrutide等）的市场表现、国内创新药企的竞争动态、减脂增肌新靶点的开发机遇，以及产业链投资机会。适合医药投资者、制药企业、研究机构和行业分析师深入了解减肥药物赛道的竞争格局、临床前景和商业化机遇。

### 蓄势待发关注减肥药赛道2ADA2025_1.md 蓄势待发关注减肥药赛道2ADA2025_1.md

本文档是减肥药赛道的核心研究报告，重点分析GLP-1受体激动剂的全球市场现状。详细介绍了司美格鲁肽和礼来替尔泊肽的销售数据与增长势头，同时深入探讨了这类药物引发的肌肉流失问题。报告强调"减重减脂但不减肌"将成为新的重磅赛道，重点介绍ActRII靶点药物Bimagrumab的临床进展，包括其在ADA2025学术会议上即将公布的联合司美格鲁肽的二期试验数据。此外，分析了礼来通过19.25亿美元收购Versanis Bio获得Bimagrumab的战略布局，以及来凯医药LAE102药物的临床合作进展。

### 蓄势待发关注减肥药赛道2ADA2025_2.md 蓄势待发关注减肥药赛道2ADA2025_2.md

本文档发布于2025年6月17日，重点分析GLP-1受体激动剂及多靶点激动剂的临床进展。详细对比了替尔泊肽作为GLP-1/GIP双靶激动剂在SURMOUNT-5三期临床中相比司美格鲁肽的优势数据（减重20.2%对13.7%）。介绍了信达生物Mazdutide（IBI362）作为GLP-1/GCG双靶激动剂的临床表现及上市进展，礼来Retatrutide三靶激动剂的开发潜力，以及国内企业众生睿创、博瑞医药等在该赛道的临床进展。文档还涉及减脂增肌靶点ActRIIA/B抗体的开发动态，以及多个企业在ADA2025科学大会上的数据公布计划。

### 蓄势待发关注减肥药赛道2ADA2025_3.md 蓄势待发关注减肥药赛道2ADA2025_3.md

本文档发布于2025年6月17日，重点分析减肥药物赛道的最新进展和投资机会。首先介绍了三靶点激动剂Retatrutide的突破性临床成果，24周减重17.5%、48周减重24.2%，创造了目前最强减肥效果。其次阐述诺和诺德以18亿美元潜在里程碑收购UBT-251的战略意义，以及国内创新药MWN-101和ZX2021的发展潜力。最后重点分析礼来口服小分子GLP-1激动剂Orforglipron的三期临床ACHIEVE-1积极结果，该药实现了7.3kg平均减重，成为首款无饮食限制的口服GLP-1受体激动剂。
