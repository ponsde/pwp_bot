# 增持维持

本目录汇集了东吴证券及相关机构发布的医药行业周报和投资分析报告，采用增持维持评级策略。内容涵盖医药股市场表现、创新药产业链动态、融资数据分析、监管政策解读及投资策略建议。报告系统追踪A股和H股医药指数、细分板块表现、上游企业财务状况、以及创新药、仿制药、生物类似物的研发进展。适合证券投资者、基金经理、医药产业研究人员及行业分析师参考，需具备医药产业和资本市场基础知识。

**核心关键词**：医药股、创新药、产业链、国产替代、融资数据、集采政策、板块估值、投资策略

## 详细描述

### 本周及年初至今各医药股收益情况_1/

本目录汇集了东吴证券研究所发布的医药行业周报数据，系统展示了医药类上市公司本周及年初至今的股票收益表现。涵盖原料药、CXO、医疗器械、生物制药、医疗服务等多个细分领域，包括沪深京港四个交易所的40余家医药企业。文档提供详细的财务指标对比、收益率分析和投资建议，适合专业投资者、基金经理、证券分析师及医药行业研究人士进行投资决策和行业跟踪。该部分数据更新频率高，是追踪医药板块短期表现和相对强弱的重要参考。

### 重视创新药早研产业链利好频出上游企业业绩兑现_2/

本目录是一份深度行业研究资料集，系统性地追踪国内创新药早期研发产业链的最新发展动态。通过融资数据分析、科研经费投入、国产替代进展、竞争格局演变、企业出海战略和具体药物研发进程等多个维度，全面展现创新药产业链上游企业如何抓住国内创新药发展机遇实现业务放量和业绩兑现。核心关键词包括：创新药、产业链上游、融资动向、国产替代、生命科学、CDMO、出海战略、竞争格局、龙头企业。适合医药行业投资者、产业分析人士、企业决策者、科研工作者和政策研究人员阅读参考。该部分重点关注产业链中游和上游的机遇，是理解创新药生态的关键资料。

### 研发进展与企业动态_3/

本目录汇总了全球制药行业的最新研发进展和企业动态追踪数据，由东吴证券研究院定期发布。涵盖已上市药物、临床试验项目、新药申报进展等多个阶段的研发信息，追踪国内外主要医药企业的创新动向。包括创新药、仿制药、生物类似物等多种药物类型的研发管线更新，以及企业并购、合作、融资等重要事项。该部分数据对于了解行业创新趋势、评估企业竞争力和预判未来产品上市机会具有重要参考价值，适合研究人员、投资者和企业战略规划人员使用。
