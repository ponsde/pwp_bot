# 研发进展与企业动态_3

本目录汇集了全球制药行业最新的研发进展与企业动态追踪数据，由东吴证券研究所定期发布。内容涵盖国际制药巨头（如Pfizer、Novartis、Novo Nordisk等）和国内领先企业的已上市药品、临床试验项目及新药申报信息。文档详细记录了药品名称、企业信息、研发阶段、剂型分类、药物靶点等关键数据，涉及肿瘤、免疫、代谢、心血管、感染性疾病等多个治疗领域。适合医药行业投资者、研究人员、监管部门及企业决策者了解行业动态、评估竞争格局和追踪创新方向。

**核心关键词**：药品上市、临床试验、新药研发、生物制品、化学药品、靶点、制药企业、注册分类

## 详细描述

### 研发进展与企业动态_3_1.md 研发进展与企业动态_3_1.md

本文档是东吴证券研究所发布的行业跟踪周报，重点呈现全球制药企业的研发进展与上市药品动态。汇总了Neurelis、Astellas Pharma、Abbott、Pfizer、Novartis、Novo Nordisk等国际制药巨头的已上市药品信息，涵盖神经、肿瘤、代谢等多个治疗领域。每条记录详细列示药品名称、上市状态、产品编号、剂型、药品类别及作用靶点。所列药品均已上市，涉及片剂、注射剂、栓剂、喷雾等多种给药途径，体现全球制药创新的最新进展。

### 研发进展与企业动态_3_10.md 研发进展与企业动态_3_2.md

本文档展示了多家国际制药企业已上市药品的详细信息，包括Neurelis、Astellas Pharma、Abbott、Pfizer、Eli Lilly、Novartis等全球知名公司的产品。采用表格形式组织数据，记录产品名称、批准状态、注册号、剂型、药物类别、分类等级和作用靶点等关键信息。涵盖地西泮鼻喷雾剂、恩扎卢胺软胶囊、司美格鲁肽片等多个已上市创新产品，涉及神经、肿瘤、代谢等多个治疗领域，便于查阅和对比分析。
