# 国产新药临床数据分析_23_DLL3

本目录汇集了2024年初国产新药临床数据分析的系列周报和专题报告，重点追踪DLL3靶点及相关创新药物的研发进展、临床试验动态和上市审批状态。涵盖内容包括生物制品、化学药品、仿制药、生物类似物等多个药物类别，涉及肿瘤免疫、感染病、心血管、呼吸系统等多个治疗领域。文档采用表格形式系统展示药物代码、靶点、临床阶段、申报编号等关键数据，为医药行业投资者、研究机构、制药企业和监管部门提供新药研发动态参考。

### 国产新药临床数据分析_23_DLL3_1.md 国产新药临床数据分析_23_DLL3_1.md

本周报重点记录2024年1月初国内外制药企业的新药审批和上市进展。主要亮点包括星汉德生物的HPV特异性TCR工程化T细胞疗法SCG142获得IND批准、阿斯利康奥拉帕利获批BRCA突变早期乳腺癌新适应症、GSK美泊利珠单抗用于慢性鼻窦炎伴鼻息肉、吉利德首款长效HIV衣壳抑制剂来那帕韦钠获批等。文档详细介绍了各药物的作用机制、适应症范围和临床意义，涉及肿瘤、感染病、呼吸系统等多个医学领域。
