# 期间融资融券情况23

## 详细描述

### 期间融资融券情况23_1.md 期间融资融券情况23_1.md
医药行业周报的融资融券统计表，汇总2025年9月8-12日期间54家医药上市公司的临时股东大会信息。涵盖深交所、上交所、北交所三个交易所，包括制药、医疗器械、医疗服务等多个细分领域。所有公司均在9月15-16日召开临时股东大会。
