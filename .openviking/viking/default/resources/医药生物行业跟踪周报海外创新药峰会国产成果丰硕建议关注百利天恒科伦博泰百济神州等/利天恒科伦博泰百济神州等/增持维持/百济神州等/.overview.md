# 百济神州等

本目录汇集了东吴证券研究所发布的医药健康产业系列周报和财务数据分析报告。内容涵盖医药生物、医疗器械、诊断试剂、CRO/CDMO等多个细分领域的上市公司财务指标、股票表现和行业动态。包括原料药、IVD、医疗耗材、仿创药、科研服务等产业链各环节的跟踪数据，重点关注百济神州、药明康德、恒瑞医药等龙头企业。适合医药行业投资者、基金经理、证券分析师和产业研究人员进行投资决策、行业对标和企业财务评估。

## 详细描述

### 百济神州等_1.md 医药行业周报与投资组合

本文档是东吴证券医药行业周报，重点追踪医药健康产业各细分领域的上市公司投资机会。主要内容包括五大领域跟踪：原料药领域涵盖奥锐特、千红制药、华海药业等；IVD领域包括亚辉龙、新产业、安图生物；医疗耗材领域有惠泰医疗、大博医疗、三友医疗；仿创药领域涉及人福医药、恩华药业、仙琚制药等；科研服务领域包括药康生物、奥浦迈、百普赛斯。核心内容为建议关注组合表，列示30余家上市公司的关键财务指标，包括市值、EPS、归母净利润及本周涨跌幅。重点公司包括恒瑞医药、科伦药业、百济神州、药明康德等龙头企业。

### 百济神州等_2.md 核心财务指标对标分析

本文档是医药生物行业上市公司财务数据对比分析表，汇总了包括百济神州、药明康德、丽珠集团、科伦药业等15家医药上市公司的核心数据。涵盖股票代码、每股收益（EPS）、归母净利润、本周涨跌幅和市值等多个维度，数据跨越2022年至2024年预期，提供历史表现与未来预测的对比。表格显示了不同企业的盈利能力差异，部分公司如百济神州出现亏损状态，而药明康德、丽珠集团等保持较好盈利。市值范围从30亿元至2732亿元不等，反映了企业规模的显著差异。

### 百济神州等_3.md 多领域行业跟踪数据

本文档是东吴证券研究所发布的行业跟踪周报，呈现医药生物行业上市公司的财务数据和股票表现分析。包含多个医药细分领域的上市公司数据，涵盖CRO/CDMO企业（康龙化成、昭衍新药等）、生物制药公司（海尔生物、诺唯赞等）、疫苗生产商（智飞生物、万泰生物等）、医疗服务机构（爱尔眼科、通策医疗等）和药品零售连锁（大参林、老百姓等）。数据表格展示了各公司的股票代码、多个时间周期的财务指标和市值信息，标注星号的公司表示存在特殊关注点。

### 百济神州等_4.md 医药健康产业财务对标表

本文档汇总了包括百济神州、康龙化成、海尔生物、智飞生物、康泰生物等25家医药生物及医疗服务企业的关键财务指标。涵盖股票代码、公司名称及多个时期的财务数据，展示了这些企业在不同年份的营收、利润、增长率等核心指标。其中智飞生物、三星医疗、康龙化成等公司的数据规模相对较大。主要用于投资分析、行业研究和企业对标，帮助投资者快速了解医药健康产业的竞争格局和财务表现。

### 百济神州等_5.md 医疗器械板块行业跟踪

本文档是东吴证券研究所发布的医疗健康行业跟踪周报，呈现医疗器械、医药生物等上市公司的财务数据对比分析。包含多个板块的上市公司数据，涵盖医疗器械企业（迈瑞医疗、联影医疗、南微医学等）、诊断试剂公司（安图生物、新产业等）、医药制造企业（华东医药、片仔癀、华润三九等）。表格展示了各公司的关键财务指标，包括营收、利润、增长率及市值等多维度数据，涵盖沪深京港四地交易所上市公司。

### 百济神州等_6.md 医疗器械诊断企业财务对比

本文档汇总了20余家医疗器械、医药生物及诊断企业的关键财务指标。涵盖的企业包括联影医疗、华东医药、爱美客、安图生物、南微医学等行业代表性公司。表格展示了各公司的营收、利润、增长率等多维度财务指标，部分公司标注星号表示特殊分类。数据涵盖多个会计期间，便于纵向和横向比较分析，主要用于医疗健康产业的投资分析、行业研究和企业对标。

### 百济神州等_7.md 百济神州专题分析与行业周报

本文档是东吴证券研究所发布的医药行业周报，重点追踪生物医药企业动态。首先呈现多家医药上市公司的财务数据对比表，涵盖贵州三力、马应龙、悦康药业等企业的估值和盈利指标。其次详细分析第43届摩根大通医疗健康年会信息，介绍近30家中国生物医药公司参展情况，重点阐述百济神州的最新进展，包括泽布替尼全球销售额突破6.9亿美元、经营现金流首次转正、2025年预期实现经营利润为正等关键业绩指标，以及血液肿瘤和实体瘤领域的多个临床催化剂。
