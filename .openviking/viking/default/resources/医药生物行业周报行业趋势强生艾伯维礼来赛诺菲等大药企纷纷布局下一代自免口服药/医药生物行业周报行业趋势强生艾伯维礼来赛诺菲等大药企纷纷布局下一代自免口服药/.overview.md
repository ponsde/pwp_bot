# 医药生物行业周报：大药企布局下一代自免口服药

本周报是2025年3月发布的医药生物行业专题分析文档，重点聚焦全球大型制药企业在自身免疫疾病口服药物领域的战略布局。文档详细分析了强生、艾伯维、礼来、赛诺菲等行业龙头通过收购和商业合作开发的创新口服自免药物，包括口服环肽IL-23分子、口服小分子IL-17、口服IRAK4降解剂等前沿产品。报告揭示了驱动这一趋势的核心因素：患者对口服给药方式的强烈需求、现有口服自免药物的有效性和安全性缺陷，以及口服药物更容易进入一线治疗市场。同时提供了2025年3月第二周医药生物板块市场表现分析和多个细分领域的投资标的推荐。适合医药行业投资者、制药企业从业人士及行业研究人员参考。

**核心关键词**：自身免疫疾病、口服药物、大药企并购、IL-23、IL-17、IRAK4、医药投资评级、行业趋势

本文档是2025年3月16日发布的医药生物行业周报，由专业研究机构出版，为医药行业投资者和从业人士提供权威的行业趋势分析。

**核心内容**：
- 详细记录了强生、艾伯维、礼来、赛诺菲等全球制药巨头在下一代自免口服药领域的具体布局案例
- 分析了企业通过收购和商业合作开发的创新产品，包括口服环肽IL-23分子、口服小分子IL-17、口服IRAK4降解剂等
- 深入阐述了驱动这一趋势的三大核心因素：
  - 患者对口服形式的强烈需求（50-70%患者未使用最新疗法，30%因注射方式问题放弃治疗）
  - 现有口服自免药物的有效性和安全性不足
  - 口服药物更容易进入前线治疗市场
