# 四维联动25Q1业绩超预期25年公司有望迎经营拐点

本目录包含信达证券关于广誉远公司2025年第一季度业绩表现的深度研究报告。报告重点分析了公司在"四维联动"战略驱动下的业绩超预期表现，以及2025年经营拐点的预期。核心内容涵盖财务数据对比、经营质量评估、战略实施效果分析等方面。报告指出公司2025Q1实现营收同比增长21.48%、净利润同比增长53.94%，扭转了2024年的颓势。适合投资者、分析师及医药行业关注者深入了解广誉远公司的经营现状和发展前景。
