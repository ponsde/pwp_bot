# 增持维持

本目录是东吴证券研究所发布的医药行业投资分析系列文档，采用增持维持评级体系，为医药行业投资者提供深度的市场分析和投资建议。内容涵盖医药股超额收益潜力评估、小核酸药物等创新领域进展、各细分板块估值分析、个股动态追踪以及药品注册分类标准等多个维度。文档包含详细的数据表格、市场表现分析、配置建议和政策风险评估，适合医药行业投资者、基金经理、证券分析师、医药企业研究人员和从业者参考。**核心关键词**：医药股、超额收益、创新药、小核酸药物、CXO、生物制品、估值分析、临床试验、集采降价、医保政策。

## Detailed Description

### 增持维持/

本目录是东吴证券研究所发布的医药行业周报系列文档的核心集合，采用增持维持评级，为投资者提供医药板块的深度分析和投资建议。主要聚焦于2025年医药股的超额收益潜力、小核酸药物等创新领域的最新进展、各细分板块（包括创新药、生物制品、CXO等）的估值分析和具体标的选择。文档包含详细的数据表格、市场表现分析、配置建议和风险提示，通过系统性的跟踪和评估，帮助投资者把握医药行业的投资机会。适合医药行业投资者、基金经理、证券分析师及医药产业研究人员使用。

### 年哪些医药股会取得显著超额收益_1_2025/

本目录是东吴证券研究所发布的2025年医药行业投资分析专题，系统性地追踪和评估医药上市公司的超额收益潜力。内容涵盖医药股票估值分析、产品管线追踪、临床试验进展、新药申报动态等多个维度，深入分析各类医药企业（创新药企、生物制品企业、CXO服务商等）的投资价值。通过对比不同细分领域的增长潜力、盈利能力和估值水平，为投资者提供全面的医药行业投资决策参考。核心关键词包括医药股、超额收益、创新药、生物制品、临床试验、产品管线、靶点分析等。适合医药行业投资者、基金经理、证券分析师及医药产业研究人员使用。

### 行情回顾_5/

本目录汇集了东吴证券研究所发布的医药行业系列周报和监管参考指南，涵盖医药板块市场表现分析、个股动态追踪、行业风险评估以及药品注册分类标准。文档通过估值对比、子板块跟踪、个股排名和政策风险分析，为医药行业投资者、基金经理、分析师和企业管理层提供数据驱动的市场洞察和投资决策参考。重点关注医药板块的周度表现、集采降价政策影响、医保谈判动态、创新药审批进展等热点问题。核心关键词包括：医药估值、板块表现、集采降价、医保政策、药品注册分类、创新药、生物制品。

### 附录_7/

本附录汇集了药品分类标准和投资评级体系的参考资料，主要涵盖生物制品和中药制剂的分类定义、新药申报标准以及证券投资评级规范。文档来源于药品审评中心（CDE）和东吴证券研究报告，为制药企业研发人员、监管部门、投资分析师和行业从业者提供专业的分类指导和投资参考。核心内容包括改良型生物制品、已上市生物制品、创新中药制剂、改良型中药新药等多个分类维度，以及相对基准投资评级体系和风险提示机制，帮助用户准确理解药品分类标准和投资评级含义。
