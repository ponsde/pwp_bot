# KEYINDEXTBL

本目录包含一份关于片仔癀股份有限公司的完整证券研究报告，由诚通证券分析师李志新撰写，发布于2025年5月15日。报告系统分析了这家百年中医药企业的业务结构、财务表现、原料供应链、产品竞争力和投资价值。核心内容涵盖片仔癀和安宫牛黄丸两大产品线的市场分析、成本结构、提价策略、毛利率变化，以及天然麝香和牛黄等关键原料的供需平衡问题。报告还提供了2025-2027年的详细财务预测、估值分析和投资评级，适合证券投资者、医药行业分析师和企业决策者参考。

## 详细说明

### KEYINDEXTBL_1.md KEYINDEXTBL_1.md
首次覆盖研究报告的核心篇章，介绍片仔癀作为百年国药瑰宝的历史背景、上市地位和国资控股结构。重点阐述两个关键平衡机制：终端产品的量价平衡（累计提价超20次，2019-2024年销售额年复合增速19.5%）和核心原料的供需平衡（2024年末原料库存32.1亿元）。强调安宫牛黄丸作为第二增长曲线的潜力，2021-2024年年复合增速达43.6%。

### KEYINDEXTBL_10.md KEYINDEXTBL_2.md
提供盈利预测、估值分析和投资评级的关键页面。预测2025-2027年营业收入分别为117.8/128.8/140.8亿元，归母净利润分别为31.7/37.2/43.2亿元，毛利率逐年提升至47.9%。对标同仁堂、云南白药等可比公司，片仔癀PE倍数为40倍，给予"推荐"评级。包含四大风险提示。

### KEYINDEXTBL_12.md KEYINDEXTBL_4.md

### KEYINDEXTBL_17.md KEYINDEXTBL_9.md
2019-2024年财务表现分析。营业收入从57.2亿元增长至107.9亿元，年复合增速13.5%；归母净利润从13.7亿元增长至29.8亿元，年复合增速16.7%。毛利率维持在45%左右，但近年因中药材原料涨价有所下滑。

### KEYINDEXTBL_18.md KEYINDEXTBL_10.md
2019-2024年财务业绩和业务结构分析。医药工业分部2024年营收达56.6亿元，同比增长17.9%，五年复合增速20%。毛利率从2021年的78.4%下降至2024年的64.7%，受上游中药材原料涨价影响。

### KEYINDEXTBL_19.md KEYINDEXTBL_11.md
三大业务板块的经营表现和股权结构。医药工业分部中肝病用药占93.8%、心脑血管用药占5.0%。医药商业分部营收从2020年28.4亿元增长至2023年42.1亿元。国资绝对控股56.51%。

### KEYINDEXTBL_2.md KEYINDEXTBL_12.md
核心产品的成本结构与定价机制分析。两大主产品的直接材料占比均超96%。天然牛黄价格从2023年1月的65万元/公斤上涨至2024年3月的165万元/公斤。片仔癀2020-2024年累计提价43.4%。

### KEYINDEXTBL_20.md KEYINDEXTBL_13.md
核心产品的毛利率变化趋势和提价策略。片仔癀毛利率从82.0%下降至67.9%，安宫牛黄丸毛利率从50.7%大幅下滑至14.2%。片仔癀自2003年上市以来超过20次提价，内销出厂价从2004年的125元/粒提升至目前600元左右。

### KEYINDEXTBL_21.md KEYINDEXTBL_14.md
肝病用药产品价格调整历史及销售业绩分析。2019-2024年年复合增速19.5%。2023年实现最大幅度提价170元，价格因素对销售增长贡献显著。2021-2024年产销两旺，库存持续低位。

### KEYINDEXTBL_22.md KEYINDEXTBL_15.md
产品库存、销售及核心原料供应链分析。2024年末库存仅14.1万盒，不足一个月销量。国家对天然麝香实行严格配额制管理，片仔癀作为首批获准使用天然麝香的五家企业之一，拥有深厚的准入壁垒。

### KEYINDEXTBL_23.md KEYINDEXTBL_16.md
天然麝香和牛黄原料供应链管理分析。片仔癀2024年肝病用药销售情况及原料消耗量推算（麝香796.5kg、牛黄1327.5kg）。全国林麝养殖规模与麝香供需缺口（年需求1000kg对比年产200kg）。公司在陕西、四川、西藏等地建立林麝养殖基地。

### KEYINDEXTBL_24.md KEYINDEXTBL_17.md
中医药企业产品认证和原料战略分析。片仔癀2024年末原料库存达32.1亿元，其中麝香和牛黄占比约29.5亿元，同比2022年增长118%。安宫牛黄丸市场规模超60亿元，零售市场占83.3%，双天然产品市场占比过半。

### KEYINDEXTBL_25.md KEYINDEXTBL_18.md
安宫牛黄丸产品市场分析。2019-2023年零售市场规模从6.6亿元增长至26.2亿元。超过120家企业拥有生产批文，但仅五家获得双天然产品资质。北京同仁堂占据绝对优势，2021年市占率达56.4%。片仔癀龙晖药业销售快速增长，年复合增速43.6%。

### KEYINDEXTBL_26.md KEYINDEXTBL_19.md
心脑血管用药产品的财务分析和原料供应研究。2022-2024年年均产量维持在95万盒左右，销售单价从204.4元/盒上升至342.2元/盒，毛利率大幅下滑至14.2%。详细阐述牛黄原料的进口政策及四种来源类型对比。

### KEYINDEXTBL_27.md KEYINDEXTBL_20.md
牛黄产品分类及市场分析参考资料。对比天然牛黄、体内培植牛黄、体外培植牛黄和人工牛黄的特征与应用。2025年4月国家药监局发布牛黄进口试点政策，允许符合条件的进口牛黄用于中成药生产，试点期限两年。
