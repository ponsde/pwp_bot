# BaoBiaoZhaiYao

本目录包含东阿阿胶公司2025年半年度业绩分析报告及相关财务预测数据。文档涵盖业绩简评、经营分析、盈利预测、估值评级、财务报表预测、投资评级统计等核心内容。主要展示公司上半年收入30.51亿元同比增长11.02%、归母净利润8.18亿元同比增长10.74%的成果，以及未来三年的盈利预测和财务健康状况评估。适合投资者、证券分析师、财务管理人员和企业管理层进行投资决策参考和财务评估。

### BaoBiaoZhaiYao_1.md BaoBiaoZhaiYao_1.md

本文档是东阿阿胶公司2025年半年度业绩分析报告摘要，由证券研究机构发布。包含四个核心部分：业绩简评展示上半年收入30.51亿元（同比增长11.02%）、归母净利润8.18亿元（同比增长10.74%）；经营分析强调核心产品阿胶稳健增长、应收账款显著改善、现金流大幅提升，以及实施高比例中期分红（派息8.17亿元）；盈利预测预计2025-2027年归母净利润分别为18.3亿、21.2亿、24.3亿元，维持增持评级，对应PE估值为18/16/14倍；风险提示涵盖产品单一、市场竞争、原材料价格波动等因素。目标受众为投资者、分析师及企业管理层。
