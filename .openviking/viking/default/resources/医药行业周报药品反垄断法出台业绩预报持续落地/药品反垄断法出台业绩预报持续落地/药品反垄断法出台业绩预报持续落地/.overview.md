# 药品反垄断法出台业绩预报持续落地

本目录是一份系统性的医药行业研究资料库，汇集了2025年1月20-24日期间的医药行业周报、投资分析、市场数据和政策解读。核心主题围绕药品反垄断法出台对医药产业的影响，以及上市公司业绩预报的持续落地情况。内容涵盖投资策略、二级市场表现、融资动态、股权变动、新药审批进展等多个维度，适合医药行业投资者、证券分析师、基金经理和产业研究人员参考。

## 详细说明

### 药品反垄断法出台业绩预报持续落地_1.md 投资策略与组合推荐
医药行业投资分析报告，提出创新药出海、医疗器械国采、红利投资三大主线。推荐包括恒瑞医药、英科医疗等9只重点个股，涵盖A股主板、科创板及港股市场的增长、稳健、创新三类投资组合。

### [2-10] 周报系列数据汇总
连续的医药行业周报，追踪2025年1月20-24日期间的市场表现。包括港股恒生医疗保健指数上涨2.7%、A股医药指数年初下跌3.75%等关键数据，PE-TTM估值25.4倍，医疗研发外包子板块表现最佳。

### [11-12] 个股涨跌与融资融券
详细统计224家上涨、256家下跌的个股表现，健友股份领涨（+12.6%），*ST大药垫底（-22.58%）。融资买入前五名为药明康德、恒瑞医药、鲁抗医药等，反映资金流向动向。

### [13-14] 股东大会与会议公告
汇总38家医药上市公司未来三个月的临时股东大会计划，涵盖康美药业、金域医学、华大基因等企业，会议时间跨度1月27日至4月27日，2月份最为集中。

### [15-16] 定增融资与重组并购
追踪38家医药企业的定增融资进展，涉及华海药业、三友医疗等，方案进度分为证监会通过、股东大会通过、董事会预案三个阶段。重组并购方面37家企业已获证监会批准或处于股东大会阶段。

### [17-20] 限售解禁与股权变动
统计18家医药企业的限售股解禁计划（1月至4月），涉及首发原股东、股权激励、定向增发等多种类型。股权质押排名前20企业中，尔康制药、珍宝岛、海王生物质押比例超40%，反映融资压力。

### 药品反垄断法出台业绩预报持续落地_28.md 新药审批与研发动态
记录布洛芬缓释胶囊获批、卡马西平缓释片获FDA批准、乙型肝炎病毒核酸检测试剂盒获证等重要进展，涉及JK07临床试验、疫苗临床研究终止等研发动态。

### [22-30] 历史研究报告库
汇集2016-2024年间的医药行业深度研究，涵盖眼科医疗、疫苗生物、中成药、医疗器械、CDMO、创新药等细分领域。重点关注创新药出海、集采政策影响、医保谈判、GLP-1降糖药、医疗服务等热点主题。

### 药品反垄断法出台业绩预报持续落地_9.md 投资评级说明与风险声明
西南证券发布的报告规范性声明，详细定义公司评级（买入、持有、中性、回避、卖出）和行业评级标准，强调信息来源合法合规，投资者需自行承担风险。
