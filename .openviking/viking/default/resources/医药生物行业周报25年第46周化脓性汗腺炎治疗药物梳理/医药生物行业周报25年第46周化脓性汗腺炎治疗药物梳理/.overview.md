# 医药生物行业周报25年第46周化脓性汗腺炎治疗药物梳理

本文档是2025年第46周医药生物行业周报，聚焦化脓性汗腺炎（HS）治疗药物的全景梳理。内容涵盖行业周度表现分析、疾病诊疗现状、治疗方案演进历程、上市药物对标及在研管线布局等多个维度。文档详细阐述了从传统抗菌疗法到生物制剂的治疗进展，重点介绍TNF抑制剂、IL-17靶点产品及新兴靶点（IL-1β、IL-36R、JAK1、BTKi等）的研发动态。同时提供医药板块投资数据、重点上市公司财务预测及估值参考，适合医药行业投资者、证券分析师、医药企业从业人士及产业研究人员阅读。

**关键词**：化脓性汗腺炎、生物制剂、TNF抑制剂、IL-17靶点、医药投资、行业周报、在研管线
